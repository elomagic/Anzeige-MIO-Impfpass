# Prototyp für die Anzeige des MIO Impfpass

<br/>
In den ersten Iterationen unseres Projektes wird ein Prototyp für die Anzeige des **MIO Impfpass 1.1.0** entwickelt.
Künftig sollen Anzeigen für alle MIOs folgen.
Der Prototyp umfasst aktuell eine interaktive JavaScript Single-Page-Application mit ReactJs, welche MIO-Daten aus statischen XML-Dateien einliest.
Für das Einlesen der MIO-Daten wird intern der MIO Parser genutzt, die Darstellung nutzt Bootstrap.

Den aktuellen Stand des Prototypen finden Sie in unserem [Github Repository](https://github.com/mio42-GmbH/Prototyp-MIO-Anzeige-Impfpass-JSSPA)
<br/>

## Aussehen und Funktionalität des Prototypen

<br/>
Die Nutzererfahrung der Anwendung wurde im Laufe erster Testungen mit Mediziner:innen in mehreren Iterationen weiterentwickelt.
Das Design ist neutral gehalten und wird künftig leicht konfigurierbar sein, um optisch an verschiedene Systeme angepasst werden zu können.
